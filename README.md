# Flappy Bird Game (C++ / SFML)

## Description
A Flappy Birdâ€“style 2D game developed in C++ using the SFML library.
The game includes multiple levels, sound effects, scoring, and limited tries per level.

## Features
- Bird movement with gravity and flap mechanics
- Randomly generated pipes
- Collision detection
- Sound effects (flap, score, game over)
- Level system with timer
- Menu, win, and game over screens

## Controls
- Space / Page Up: Flap
- Page Down: Move down
- Enter: Start / Retry
- Escape: Exit

## Requirements
- C++
- SFML (Graphics, Audio)
- Windows OS

## How to Run
1. Install SFML
2. Clone the repository
3. Build using a C++ compiler linked with SFML
4. Run the executable
